Pour démarrer le programme :

	1. Déplacez vous à la racine du dossier du programme dans le terminal
	2. Tapez la commande suivante dans le terminal pour exécuter le programme : pyhton3 main.py

Il est possible de préciser le fichier d'entrée via l'option -i et de demander un fichier de sortie via -o, par exemple :
  python3 main.py -i fichier.txt -o sortie.py
